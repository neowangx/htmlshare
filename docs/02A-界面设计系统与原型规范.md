## 1. 设计方向（按 Q6 可在门禁前免费更换）

htmlshare 的界面 = **产出的阅读页本身**（模板×风格）+ 门禁页。方向定位：**内容优先的编辑排版**——页面存在感让位于内容，交互组件只为降低认知负担（docs/02 §1 准则的视觉化）。四种风格是同一骨架上的四套 token 主题，不是四种布局。

## 2. Design Tokens（缺省值 = clinical 风格；其余风格按 §3 覆写）

```
排版   正文 16px/1.65；字号阶梯 13/14/16/20/26/32；标题字重 650，正文 400
       行长上限 72ch；中文优先栈: -apple-system,"PingFang SC","Noto Sans SC",sans-serif
       代码: ui-monospace,"SF Mono",Menlo,monospace, 14px
间距   4 的倍数体系: 4/8/12/16/24/32/48/64；区块间距 32，卡片内边距 20
圆角   卡片 10px；按钮/输入框 8px；徽标 999px
色彩   clinical(浅): 背景 #F7F8FA / 卡面 #FFFFFF / 正文 #1A2233 / 次要 #5A6472
       主色 #2456D6；语义: 成功 #1F7A4D / 警告 #B96A00 / 危险 #C23934（深色模式各配对应变量）
阴影   仅两级: 卡片 0 1px 3px rgba(16,24,40,.07)；浮层 0 8px 24px rgba(16,24,40,.12)

```
全部 token 以 CSS 变量输出（`--hs-*` 前缀），C-06/C-14 的「tokens 被实际引用」断言以此为准。

## 3. 四风格差异表（同骨架，只动 token）

| 风格 | 覆写要点 |
|------|---------|
| clinical | §2 缺省值；卡片分区感最强 |
| minimal | 无卡片无阴影，纯排版：背景=纸白 #FCFCFC，分隔靠间距与 1px 细线，主色只用于链接 |
| editorial | 标题阶梯放大（26→34），首段 leading 提升，衬线标题栈（Georgia,"Songti SC"），引用块强化 |
| darktech | 固定深色 #0E1116/#161B22，代码块占视觉主位，主色 #4C8DFF，对比度仍须 ≥4.5:1 |

## 4. 组件视觉规范（各状态）

- **TL;DR 面板**：左侧 3px 主色条 + 卡面；每条 ≤2 行；≤5 条；无内容不渲染（禁止空面板）。
- **原文⇄增强开关**：右上角胶囊双段钮；当前态实底、另一态透明；键盘 Tab 可达、Enter 切换；切换 120ms 淡入。
- **折叠 `<details>`**：summary 左侧 ▸ 旋转 90°；展开区左缩进 16px；打印全展开。
- **卡片组（行动项等）**：≥3 条同构才用；卡内「负责人」徽标 + 期限右对齐；缺项显示「未指定」灰字。
- **门禁页**：单列居中 360px；输入框 48px 高、自动聚焦；错误态红框 + 下方 13px 文案（不弹窗）；限速态禁用按钮 + 倒计时；解密进度用细进度条（静态轨）。
- **页脚**：13px 次要色单行；诚实声明可展开 details。

## 6. 防 AI 味负面清单（命中任一条，U/C 界面卡验收不过）

1. 紫蓝渐变、玻璃拟态、发光描边任意出现；
2. emoji 用作功能图标（内容里用户自己写的不算）；
3. 无信息层级的等权重卡片海；
4. 标题正文同字重同字号（层级失效）；
5. 页面出现 §2 之外的自造颜色/圆角/阴影值；
6. 深色风格纯黑底(#000)纯白字(#FFF)；
7. 装饰性动画（非状态反馈的位移/视差/循环动效）。
